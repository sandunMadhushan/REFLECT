# REFLECT v2.0 — Release Notes

## 📱 What's New in v2.0

### 🏠 Home Screen
- **Interactive bar chart** — tap any day (Mon–Sun) to filter the Recent Activity section by that specific day
- **Mixed Recent Activity feed** — now shows Goals, Reflections, and Habit completions together (not just goals)
- Bar chart height now reflects **all activity** (goals + reflections + habits combined per day)
- Tapping the same bar again closes the detail panel (toggle behaviour)

### 📖 Reflection Journal
- **Calendar icon now fully functional** — tap to open a date picker and filter reflections by any specific date
- Selected date shown in a banner with **"Showing: March 12, 2026"** style label
- **"✕ Clear" button** to remove the date filter and go back to all reflections
- Future dates are blocked from selection
- **Filter-aware empty states** — each filter tab (All time / This week / This month / Favorites) shows its own contextual message
- Switching any filter chip automatically clears the date filter

### 🎯 Goals
- **Filter-aware empty states**:
  - Active tab → "No active goals"
  - Completed tab → "No completed goals yet — Keep going!"
  - All tab → "No goals yet"

### 📊 Progress Analytics
- **Clickable day bars** — tap Mon–Sun to see a **Day Detail panel** inside the card showing:
  - 🔥 Habits completed that day (with streak count)
  - 🎯 Goals updated that day (achieved / in progress)
  - 📝 Reflections written that day (with mood)
- Today's bar is **auto-selected** on open
- Tap same bar again to **close** the detail panel
- Switching time filter chips (This Week / This Month / All Time) resets bar selection
- **"✕" close button** on the detail panel

### 👤 Profile
- **Progress Analytics** moved to its own **ANALYTICS** section (separate from Vision Board)
- Vision Board and Analytics are now two distinct sections with their own headers

### 📔 Navigation
- **Journal icon** in bottom nav bar recreated as a clean open-book vector icon

## 📲 Installation
1. Download **REFLECT-v2.0.apk** below
2. On your Android device: **Settings → Security → Allow unknown sources**
3. Open the downloaded APK and tap **Install**

> ⚠️ Minimum Android version: 7.0 (API 24)  
> ℹ️ If you have v1.x installed, you can install v2.0 over it — your data will be preserved.
